# AI Consultant by KIV

MVP веб-приложение для анализа текстовых запросов с использованием LLM.

## Цель
Создать минимально жизнеспособный AI-консультант с backend и frontend, способный принимать текст пользователя, отправлять его в языковую модель и отображать структурированный ответ.

## Стек
- Python
- FastAPI
- HTML
- CSS
- JavaScript

## Функционал
- ввод текста пользователем;
- отправка запроса на backend;
- обработка запроса через LLM API;
- вывод ответа в интерфейсе;
- базовая обработка ошибок.

## Архитектура
Frontend → Backend → LLM API → Backend → Frontend

## Запуск проекта

### 1. Создание окружения
```bash
python -m venv venv
```

### 2. Активация окружения
```bash
.\venv\Scripts\Activate.ps1
```

### 3. Установка зависимостей
```bash
pip install -r requirements.txt
```

### 4. Запуск сервера
```bash
uvicorn backend.main:app --reload
```

### 5. Открыть в браузере
```text
http://127.0.0.1:8000
```

## Ограничения MVP
- нет базы данных;
- нет веб-поиска;
- нет RAG;
- нет пользовательской авторизации;
- система работает только с текстом, переданным пользователем.
